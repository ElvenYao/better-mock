## Random.capitalize

* Random.capitalize(word)

把字符串的第一个字母转换为大写。

```js
Random.capitalize('hello')
// => "Hello"
```

## Random.upper

* Random.upper( str )

把字符串转换为大写。

```js
Random.upper('hello')
// => "HELLO"
```

## Random.lower

* Random.lower( str )

把字符串转换为小写。

```js
Random.lower('HELLO')
// => "hello"
```

## Random.pick

* Random.pick( arr )

从数组中随机选取一个元素，并返回。

```js
Random.pick(['a', 'e', 'i', 'o', 'u'])
// => "o"
```

## Random.shuffle

* Random.shuffle( arr )

打乱数组中元素的顺序，并返回。

```js
Random.shuffle(['a', 'e', 'i', 'o', 'u'])
// => ["o", "u", "e", "i", "a"]
```
