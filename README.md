[官方文档](http://lavyun.github.io/better-mock)

## 初衷

Mock.js 是一个很好的库，国内使用者也挺多，但是该库在3年前就几乎已经停止维护了，而且在当前时代下，不管从构建工具、代码质量还是社区生态上来看，Mock.js 都显得很陈旧，所以 better-mock 将会在 Mock.js 的基础上进行构建，持续修复 Mock.js 的众多issue，支持更多的新特性。
